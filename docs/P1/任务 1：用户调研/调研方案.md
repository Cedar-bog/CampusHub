# 问卷

1. 给出你希望有的板块
2. 请问你认为互助平台应该包含哪些功能？
3. 以上功能或板块中，你认为最重要的是？
4. 你认为通过学号认证注册方式如何？
5. 为了让平台活下去且维持良好氛围，你认为需要管理哪些方面？
6. 你希望平台的收费方式是？
7. 如果平台需要志愿管理员，你的应聘意愿是？
8. 你对平台建设还有哪些建议？
9. 要是平台真建成了且真投入使用了，你的使用意愿是？
10. 你在生活中还有什么需要他人参与协助的需求？
11. 你在互助平台上想要得到怎样的酬劳，你愿意接受本平台的积分吗？
12. 在互助的过程中，你认为需要与对方进行线上交流吗？
13. 你认为平台应该通过什么标准来判断一个互助任务成功或者失败？

# 访谈提纲

1. 回忆一个具体的“求助无门”场景

   ```
   请回忆一下，在过去半年里，有没有哪次你急需别人帮忙（比如取快递、借复习资料、组队参加比赛），但在微信/QQ群里发了消息却效果不好？当时你发了什么内容？等了多久？最后怎么解决的？ 如果整个过程让你最烦躁的一个点，会是什么？
   ```

2. 关于“信任”与“学号认证”的权衡

   ```
   问卷里多数同学认可学号认证。但我们想听听你的真实顾虑：如果你看到一个需要‘进宿舍帮忙取东西’的求助，对方是实名认证过的同学，和对方只是一个‘昵称+校园卡照片’，你的信任感会有多大差别？ 另外，如果认证后你的‘交易记录’（比如被投诉、爽约）会被保留，你会担心什么？
   ```

3. 模拟一次“任务成功与失败”的判断

   ```
   假设你在平台上发布‘帮忙去图书馆还书’，对方答应了。但后来书没还成，因为你没把书放在约定地点。你觉得这次任务算谁的责任？平台应该怎么判断‘成功’还是‘失败’？ 如果双方各执一词，你觉得平台需要看聊天记录、位置签到、还是拍照留证？哪种方式你觉得最公平且不麻烦？
   ```

4. 激励机制的“内心定价”

   ```
   问卷里提到‘积分’。假设你帮同学代取了一个快递（正常排队10分钟），平台给你10积分。你觉得这10积分大概值多少钱？（比如兑换一杯奶茶需要100积分） 另外，如果积分只能用来发布求助或换一些小纪念品，和积分能按比例提现（比如100积分=1元），你的参与意愿会有变化吗？为什么？
   ```

5. 线上聊 vs 直接行动的偏好
   ```
   问卷显示大家倾向线上交流。那么具体到一个‘辅导高数’的求助：你是希望双方先在平台内聊清楚‘哪道题、讲多久、对方水平如何’，还是希望平台直接提供‘约好时间地点线下见面讲’的标准化流程？ 你觉得线上聊太多会不会反而降低效率？在什么情况下你愿意跳过聊天，直接接单？
   ```
6. 作为“志愿管理员”的日常想象

   ```
   问卷里有同学愿意当志愿管理员。假设现在你就是管理员，需要处理一件小事：有人举报一条‘求代考’的求助帖，但发布者说‘我只是开玩笑’。 你会怎么判断？需要看什么证据？你会觉得处理这种事很心累吗？如果每周要花1-2小时处理类似纠纷，你更希望平台给你什么回报？（比如积分、证书、小礼品？）
   ```

7. 对“平台活下去”的商业模式接受度

   ```
   我们想长期运营这个平台。假设两种方案：A. 发布‘急需型’求助（比如1小时内要取快递）每单收0.5元服务费，普通求助免费；B. 完全免费，但首页会有驾校、留学机构的广告。 你个人更能接受哪种？或者你有更接受的方案吗？比如会员制、悬赏加速包？
   ```

8. 一个你可能没想到的功能需求
   ```
   除了问卷里提到的快递、学习、二手这些，你有没有遇到过一种‘很难归类但特别希望有人能帮忙’的事？ 比如深夜从实验室回宿舍找人语音陪聊、帮忙早起占座、甚至假装是朋友接个电话？如果有这样的功能，你觉得平台应该允许吗？为什么？
   ```
