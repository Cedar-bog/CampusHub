# 1.3 AI 对比实验 —— "AI 需求盲区挑战"

这是本阶段的**关键实验**，请严格按以下步骤执行：

**第一步：AI 生成**

- 向 AI 提供以下 Prompt（或团队优化后的版本）：

> "请为一个校园互助服务平台生成完整的需求规格说明书，平台支持学生发布和接单各类互助需求（快递代取、学习辅导、二手交易、组队匹配等），目标用户为大学生。请按 IEEE 830 标准格式输出。"

- 保存 AI 生成的完整输出

**第二步：人工调研**

- 基于问卷和访谈结果，团队独立整理需求（不参考 AI 输出）

**第三步：对比分析**

- 将 AI 生成的需求与人工调研结果对比，填写下表：

| 对比维度            | AI 生成的需求 | 人工调研发现的需求 | 分析                 |
| ------------------- | ------------- | ------------------ | -------------------- |
| AI 正确覆盖的需求   |               |                    |                      |
| AI 遗漏的需求       |               |                    | 为什么 AI 会遗漏？   |
| AI 捏造的需求       |               |                    | 这些需求真的存在吗？ |
| AI 表述有歧义的需求 |               |                    | 歧义在哪里？         |
